Proyecto realizado en Ionic, angular y capacitor. El mismo fue desarrollado como parte de una prueba tecnica para GVAM.
El objetivo principal es demostrar mis habilidades en desarrollo front-end y la arquitectura de proyectos.

Intalación y ejecución:

- Clonar el repo: git clone git@github.com:AntikTomas/gvam.git
- Cambiar de directorio: cd para cambiar de repositorio
- Instalar paquetes node: npm i
- Levantar el proyecto con ionic serve o ionic cap run android
